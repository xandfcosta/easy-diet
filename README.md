# Easy Diet
Aplicativo para cálculo e criação de refeições para facilitar 

## Requisitos
### NPM
Necessário ter o npm instalado para rodar o aplicativo

### MySQL
Necessário term o MySQL instalado. Na pasta Database tem o arquivo create_database.py que lé o arquivo database.json e envia para a table do MySQL (necessário criar a table antes)
Ou pode-se utilizar do arquivo Database.sql para importar direto pelo Workbench

### Python Uvicorn
Hospeda o backend no localhost para fazer as requisições no banco de dados
